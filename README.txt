== Description ==

Awesome Wordpress Slider is the most powerful and intuitive WordPress plugin to create sliders which was never possible before. Fully responsive, SEO optimized and works with any WordPress theme. Create beautiful sliders and tell stories without any code.


==Most powerful features==

<ul>
	<li><strong>Content slide builder</strong> - Edit your slides as you used to in shortcoder builders.</li>
	<li><strong>Totally responsive and touch friendly</strong> - Sliders can be enjoyed on any device, be it a mobile, tablet or desktop.</li>
	<li><strong>Overlay Color</strong> - This choise is for overlay, that appear as background for titles an descriptions on the slides.</li>
	<li><strong>Slideshow Speed</strong> - This speed of each slide will remain in the slieder.</li>
	<li><strong>Customizable controls</strong> - Arrows, Bullets, Autoplay, Bar, Thumbnails, Shadows</li>
	<li><strong>Navigation arrows</strong> - You can enable / disable the button of navigation (Previous/next) for the slider for the option. You have the option to specify the color and size.</li>
	<li><strong>Dynamic Slides</strong></li>
	<li><strong>Full-width responsive mode</strong></li>
	<li><strong>SEO optimized</strong></li>
	<li><strong>Totally integrated with WordPress</strong></li>
</ul>



== Installation ==

<h2>This section describes how to install the plugin and get it working.</h2>
<p>First things first, thank you for choosing our Awesome Wordpress Slider plugin!</p>


<h4>You can install the plugin one of two ways:</h4>
<h4>Install using FTP</h4>
<ol>
	<li>Unzip the awesome-wordpress-slider.zip file locally to your machine</li>
	<li>Connect to your FTP area using your preferred FTP package</li>
	<li>Upload the Awesome Wordpress Slider folder that you extracted from the zip file to the plugin folder of your WordPress installation (wp-content -> plugin)</li>
	<li>Go to plugin and activate Awesome Wordpress Slider.</li>
</ol>
<h4>Install by WordPress (recommended)</h4>
<ol>
	<li>Login to your WordPress admin area.</li>
	<li>Navigate to plugin and click the Add New button at the top</li>
	<li>Click the Upload plugin button at the top</li>
	<li>Click the Choose File button and locate the awesome-wordpress-slider.zip </li>
	<li>File on your machine and click the Install Now button </li>
	<li>On the basis the plugin installs correctly, click the Activate link.</li>
</ol>
